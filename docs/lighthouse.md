# Lighthouse report

Drop your Lighthouse screenshot(s) in this folder (e.g. `docs/lighthouse-listing.png`, `docs/lighthouse-detail.png`) and reference them from the README if you want to showcase performance/accessibility scores.

To generate one: open the deployed site (or `npm run build && npm run preview`) in Chrome, open DevTools → Lighthouse, run an audit for Performance/Accessibility/Best Practices/SEO, and save the summary screenshot here.
